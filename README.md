## 🎬 Lottie Animations - Ultra Optimizado

### **¿Por qué Lottie en lugar de GIF?**

| Característica | GIF 🐌 | Lottie JSON ⚡ |
|---------------|--------|---------------|
| **Peso** | 2-5 MB | 20-150 KB |
| **Calidad** | Pixelada, se degrada | Vectorial perfecta |
| **Escalado** | Pierde calidad | Sin pérdida, cualquier tamaño |
| **Carga** | 3-5 segundos | Instantánea < 0.5s |
| **Rendimiento** | Consume mucha RAM | Ultra ligero |
| **Animación** | Frames fijos | Smooth 60 FPS |

**💡 Lottie es hasta 20x más pequeño que un GIF equivalente**

### **Cómo agregar animaciones Lottie**

1. **Descarga** animaciones desde [LottieFiles.com](https://lottiefiles.com/)
2. **Coloca** los archivos `.json` en `/public/images/projects/`
3. **Actualiza** la ruta en `Projects.tsx`:

```tsx
{
  id: 1,
  image: '/images/projects/tu-animacion.json', // ← Ruta del archivo JSON
  // ... resto de configuración
}
```

**Recursos recomendados:**
- [LottieFiles - Marketplace](https://lottiefiles.com/featured)
- [After Effects + Bodymovin Plugin](https://github.com/airbnb/lottie-web)
- [Lottie Animation Community](https://lottiefiles.com/community)

## 🐛 Troubleshooting

### Error: PowerShell no ejecuta npm

**Solución:**
```powershell
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```

O usa **Git Bash** o **CMD** en su lugar.

### Las animaciones Lottie no se ven

1. Verifica que el archivo `.json` esté en `/public/images/projects/`
2. Asegúrate de que la ruta sea correcta: `/images/projects/nombre.json`
3. Revisa la consola del navegador para errores

### El tema dark/light no cambia

1. Verifica que `ThemeProvider` esté en `layout.tsx`
2. Limpia caché del navegador
3. Revisa que `globals.css` tenga las variables CSS
